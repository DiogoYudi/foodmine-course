## Installation

1. Clone the repository:

```bash
git clone https://github.com/DiogoYudi/foodmine-course.git
```

2. Open the repository in any IDE

3. Install npm
   
```bash
npm install
```

## Usage

1. Start the application
```bash
ng serve
```
or
```bash
ng s
```
2. The API will be accessible at http://localhost:4200/
